The folder "paratikasamples" contains samples for various integration models
In order to test the JSP pages properly, please copy the whole folder "paratikasamples" into a Tomcat distribution in "webapps" folder. 
After running tomcat, you will be able to browse the JSP pages via

- http://localhost:8090/paratikasamples/jsp/session/sessiontoken.jsp
- http://localhost:8090/paratikasamples/jsp/directpost/directpost.jsp
- http://localhost:8090/paratikasamples/jsp/session/hpp-redirection.jsp
- http://localhost:8090/paratikasamples/jsp/api/sale-preauth.jsp

In order to test directpost, hpp-redirection or sale-preauth, you first need to create a session token.
